# Arduino Keypad
4x4 Keypad for Arduino Uno R3. Featuring a buzzer and a RGB LED.

![alt text](https://github.com/PyhaMarkus/arduino-keypad/blob/master/pictures/keypad_bb.png "Arduino Keypad")

---

### Parts used:
| Quantity | Part name                             |
| -------- |:-------------:                        |
| 1        | Arduino Uno R3                        |
| 1        | 4x4 Keypad                            |
| 1        | Passive buzzer                        |
| 1        | RGB LED                               |
| 1        | 100 ohm resistor                      |
| 3        | 220 ohm resistor                      |
| 1        | A small breadboard                    |
| A lot    | Jumper wire                           |
